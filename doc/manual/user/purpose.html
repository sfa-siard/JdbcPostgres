JdbcPostgres - Users' Manual - Purpose

Purpose of the Project

JdbcPostgres was developed as part of the SIARD project. It implements the JDBC interface for Oracle databases. It presents a standardized (JDBC 4.1, SQL:2008) JDBC interface. However, it has only been tested to the extent, that was used in the SiardCmd project.
