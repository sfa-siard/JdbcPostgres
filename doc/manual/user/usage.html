JdbcPostgres - Users' Manual - Usage

Using the Binaries of the Project

This JDBC driver for DB/2 is to be used like any other JDBC driver. It is recommended, that the static method Db2Driver.register() is used to register it with the DriverManager.

The support for executing SQL is based on a parser for SQL:2008 and is limited to the single-table query functionality needed for SIARD Suite 2.1.

A GUI for JdbcPostgres

JdbcPostgres is distributed as a collection of JAR files without a graphical user interface. However, the versatile free application SQuirreL SQL Client can be used to make use of it.

Handling of Range Types

Range types are mapped to UDTs with the following fields: range_start: <type of the range> range_end: <type of the range> range_signature: char(2) (must be "[)", "()", "(]", or "[]")

Handling of LOBs

Two special domain types "public.blob" and "public.clob" are created in the database. They hold OIDs, which point to large objects.

Handling of Domains

Domain types are mapped to DISTICT types and vice versa.

Handling of Enum Types

Enum types are mapped as DISTICT types with base type VARCHAR.

Handling of multidimensional array

Multidimensional arrays are mapped to one-dimensional arrays.
